# 🛡️ CyberShield - Cyber Security Awareness Toolkit

## 📌 Project Overview

CyberShield is a beginner-friendly Cyber Security Awareness Toolkit developed using HTML, CSS, and JavaScript. The project helps users learn important cyber security concepts through interactive tools and educational content.

## ✨ Features

* 🔐 Password Strength Checker
* 🔑 Password Generator
* 🌐 URL Safety Checker
* 🧠 Cyber Security Quiz
* 🛡️ Cyber Security Tips
* 📱 Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

* Home Page
* Password Tools Page
* URL Checker Page
* Quiz Page
* Tips Page
* About Page

## 🎯 Purpose

The purpose of this project is to promote cyber security awareness and help users understand safe online practices.
