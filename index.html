Cyber Security Awareness Toolkit

CyberShield is a beginner-friendly cyber security awareness toolkit developed using HTML, CSS and JavaScript.

The toolkit includes a Password Strength Checker and Password Generator to help users create stronger passwords, a URL Safety Checker to identify suspicious links, an interactive Cyber Security Quiz to test awareness, and a collection of essential Security Tips for safer online practices.

Its goal is to promote cyber security awareness through simple, practical and easy-to-use tools.
